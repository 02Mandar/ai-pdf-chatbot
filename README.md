# ai-pdf-chatbot
Built an AI-powered chatbot capable of answering questions from uploaded PDF documents using Generative AI and semantic search. Implemented document chunking, embeddings, and FAISS vector database for accurate information retrieval. Integrated LangChain with Llama 3 to generate context-aware responses through an interactive Streamlit interface.

## Features
- PDF Question Answering
- Semantic Search
- Context-Aware AI Responses

## Tech Stack
Python, LangChain, FAISS, Streamlit, Ollama
